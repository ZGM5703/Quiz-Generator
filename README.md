Quiz Generator with Claude API

Paste any notes or type a topic (e.g. "OSI model", "Python loops", "Philippine history")
Three difficulty levels — Easy (recall), Medium (understanding), Hard (application)
5 questions with A/B/C/D options — click to answer, instant right/wrong feedback
Each question reveals an explanation after you answer
Progress bar + live score tracker
Final results with a performance message
Retry same quiz or generate a new one
